# Wildlife Conservation Non-Profit - Premium HTML Template

A high-performance, responsive, and SEO-ready website template designed for Wildlife Conservation organizations and Non-Profits.

## Features
- **Modern Design**: Built with a Deep Jungle Green and Earthy Sand palette.
- **Donor Dashboard**: Included for impact tracking and virtual patrol updates.
- **Responsive**: Mobile-first design with a slide-out hamburger menu.
- **Dark Mode & RTL**: Fully supported via CSS variables and direction switching.
- **SEO Optimized**: Semantic HTML5, proper meta tags, and accessibility (WCAG 2.1 AA).

## Pages Included
- Home (variation 1 & 2)
- About Us
- Our Missions (Services)
- Nature Blog
- Contact Us
- Donor Dashboard
- Auth Pages (Login/Register)

## Customization
- **Colors**: Update `:root` variables in `assets/css/style.css`.
- **Fonts**: Modify the Google Fonts import in `assets/css/style.css`.
- **Images**: Replace files in `assets/images/` with your own assets.

## Installation
Simply open `pages/index.html` in any modern web browser or host the root directory on a web server.
